1. 제품 개요


1.1 비전
교육상품(도서, 학습지, 에듀테크, 온라인 강의 등)을 유통·판매하는 사업자가
"내 마케팅이 잘 되고 있는가? 어떻게 수정해야 하는가? 더 좋은 방법은 무엇인가?"
를 대시보드 하나로 답할 수 있는 AI 마케팅 플랫폼.

사업자가 수작업으로 3일 걸리던 시장조사·여론분석·캠페인 기획을
AI가 30분 안에 자동화하고, "다음에 할 일 3가지"까지 제안한다.


1.2 핵심 가치 제안 (v2.0 개정)
* 즉시 인사이트 — 한국어 키워드 1개 입력 → 30분 내 한국 시장 여론 분석 + 전략 권고
* 내 사업 대시보드 — 예상 매출, 캠페인 ROI, 경쟁사 대비 포지셔닝을 한 화면에
* 액션 자동화 — "이 메시지로 캠페인 보내기", "피칭 덱 자동생성", "다음 주 콘텐츠 캘린더" 제안
* 한국 시장 특화 — 네이버 쇼핑/블로그, 인스타그램, 맘카페 등 한국 교육시장 핵심 채널 자동 수집
* 데이터 기반 — 모든 의사결정에 정량적 근거 + AI 신뢰도 표시

2. 타겟 사용자 (v2.0 재정의)


2.1 주요 페르소나

페르소나 A — 교육상품 유통 담당자
	직무: 중소 교육기업 마케팅/영업 담당 (1~3년 경력)
목표: 이번 분기 매출 목표 달성, 효과적인 채널·메시지 발굴
Pain Point: 네이버·인스타 트렌드 파악에 매주 10시간+ 소요 / 경쟁사 모니터링 수작업
핵심 니즈: "지금 뭐가 뜨는지" + "우리 상품 반응이 어떤지" + "뭘 해야 하는지"
	

페르소나 B — 1인 교육 사업자
	직무: 학습지/온라인 강의 판매, SNS 마케팅 병행 (1인 사업)
목표: 월 매출 1,000만원 달성, 효율적 마케팅 채널 집중
Pain Point: 마케팅 대행사 비용 부담 / 어떤 채널이 효과적인지 데이터 없음
핵심 니즈: 저비용 자동화, 한 화면에서 성과 확인, "따라하기" 가이드
	

페르소나 C — 교육기업 대표
	직무: 교육 콘텐츠 기업 대표 (직원 10~50명)
목표: 연간 마케팅 ROI 개선, 신상품 출시 전략 수립
Pain Point: 팀별 마케팅 성과 비교 어려움 / 데이터 기반 전략 부재
핵심 니즈: 1페이지 요약 리포트, 부서별 성과 비교, 분기 트렌드 요약
	

2.2 사용 시나리오 (v2.0 신규)

시나리오 1 — "오늘의 마케팅 상황 파악" (매일 아침 3분)
	① 대시보드 접속 → 예상 매출 + 전주 대비 변화 확인
② 내 브랜드 온라인 반응 요약 카드 확인 (긍/부정 + 핵심 키워드)
③ "이번 주 할 일 3가지" 확인 → 클릭하면 상세 가이드
	

시나리오 2 — "신상품 출시 전 시장 분석" (1시간)
	① "초등 코딩 학습지" 키워드로 SignalCraft 실행
② 네이버 블로그/카페 + 인스타 + 맘카페에서 여론 수집
③ 30분 후 통합 리포트: 시장 반응 + 기회 영역 + 추천 메시지 + 경쟁사 대비
④ "이 메시지로 인스타 캠페인 초안 생성" 버튼 클릭 → 초안 자동 생성
	

시나리오 3 — "경쟁사 모니터링" (주 1회 자동)
	① 경쟁사 5곳 자동 모니터링 (주 1회 batch)
② 대시보드에 "경쟁사 동향 카드": 신상품 출시, 프로모션, 온라인 반응 변화
③ 내 포지셔닝 vs 경쟁사 시각화 (장르·가격·반응 축)

8. 비기능 요구사항 (v1.0 유지 + v2.0 추가)


* 성능: SignalCraft 결과 30분 이내 (v1.0은 1~3시간)
* 모바일: 핵심 KPI 3개(매출/반응/액션)는 모바일에서 3초 확인 (v2.0 신규)
* 보안: v1.0 유지 (helmet, CORS, Sentry, 카테고리 분리, DB 롤)
* 인프라: Railway (기존 유지) + Cloudflare DNS
* 비용: API 비용 월 $50 이하 유지 (Prompt Caching + Haiku fallback)
* 법적: v1.0 OI-01~06 유지 + v2.0 추가: 개인정보 처리 위탁계약(Naver/Instagram 크롤링 시)
