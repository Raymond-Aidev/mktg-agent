# GoldenCheck PRD (Product Requirements Document)

**문서 버전**: 1.0
**작성일**: 2026년 2월
**최종 수정**: 2026년 2월 2일

---

## 1. 제품 개요

### 1.1 제품명
**GoldenCheck** - 시니어 건강 모니터링 플랫폼

### 1.2 제품 비전
스마트폰만으로 시니어의 신체적·인지적 건강 상태를 손쉽게 모니터링하고, 조기 이상 징후를 감지하여 적시에 전문 의료 상담을 유도하는 서비스

### 1.3 핵심 가치 제안
| 가치 | 설명 |
|------|------|
| **접근성** | 특수 장비 없이 스마트폰만으로 건강 체크 |
| **조기 발견** | 미세한 건강 지표 변화를 추적하여 조기 경고 |
| **비침습적** | 사진과 음성만으로 분석, 신체 접촉 불필요 |
| **지속적 모니터링** | 트래킹을 통한 장기적 건강 추세 파악 |

### 1.4 대상 사용자
- **주 사용자**: 60세 이상 시니어
- **보조 사용자**: 시니어를 돌보는 가족, 요양보호사
- **전문 사용자**: 요양원, 주간보호센터 운영자

---

## 2. 제품 구성

### 2.1 모듈 구조

```
GoldenCheck
├── 영상 분석 모듈 (Image Analysis)
│   ├── 피부 분석 (Skin Analyzer)
│   ├── 눈 분석 (Eye Analyzer)
│   ├── 입술 분석 (Lip Analyzer)
│   ├── 손톱 분석 (Nail Analyzer)
│   └── 얼굴 종합 분석 (Face Analyzer)
│
├── 음성 분석 모듈 (Voice Analysis)
│   ├── 운율 분석 (Prosodic Analyzer)
│   ├── 발음 분석 (Pronunciation Analyzer)
│   └── 인지 평가 (Cognitive Evaluator)
│
├── 트래킹 시스템 (Tracking System)
│   ├── 건강 기록 저장
│   ├── 추세 분석
│   └── 변화 감지 및 알림
│
└── 리포트 시스템 (Report System)
    ├── 개인 건강 리포트
    ├── 추세 리포트
    └── 알림 및 권고사항
```

class HealthRecord:

    risk_level: str

## 6. 사용자 시나리오

### 6.1 시나리오 1: 일일 건강 체크

```
사용자: 70세 여성, 혼자 거주
목표: 매일 아침 건강 상태 확인

흐름:
1. 앱 실행 → "오늘의 건강 체크" 선택
2. 셀프 카메라로 얼굴 촬영 (자동 가이드)
3. 분석 진행 (2-3초)
4. 결과 확인:
   - 전체 건강 점수: 85/100
   - 주요 지표: 모두 정상 범위
   - 어제 대비: 변화 없음
5. "좋아요! 오늘도 건강합니다" 메시지 표시
```

### 6.2 시나리오 2: 인지 기능 모니터링

```
사용자: 75세 남성, MCI 의심 진단 후 추적 관찰 중
목표: 주 2회 인지 음성 평가

흐름:
1. 앱 실행 → "음성 인지 검사" 선택
2. 화면에 문장 표시: "건강을 위해서 매일 운동하는..."
3. 문장을 소리 내어 읽음 (녹음)
4. 분석 진행 (3-5초)
5. 결과 확인:
   - 전체 점수: 72/100
   - 위험 수준: LOW_RISK
   - 변화: 지난주 대비 말하기 속도 5% 감소
6. 권고: "다음 주 재검사를 권장합니다"
7. 결과 자동 저장 및 추세 그래프 업데이트
```
